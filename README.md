# g2pt_workshop_2026 - Show Me Your Repo — Workshop
This is the repository for workshop Geek Girls Portugal Conference 2026 - "Show Me Your Repo. I’ll show you mine."
Welcome to the hands-on GitHub workshop.


## **Goal**

Understand why standards matter


## **Your Task**

1. Go to your assigned folder (e.g. `group_1/`)
2. Open the Python script
3. Follow the instructions inside the file
4. Create a branch:

   ```bash
   git checkout -b task/g_<your-name>
   ```
5. Make your changes
6. Commit with a meaningful message
7. Push your branch
8. Open a Pull Request


## **Important**

* Do **not** push directly to `main`
* Use a clear commit message
* Fill in the PR description properly


## **What will happen**

You will likely:

* Write unclear commit messages
* Forget to describe your PR
* Miss important details

That’s expected — we will fix it together.


## **Repository Structure**

```
group_1/
group_2/
...
group_30/
```

Each folder contains:

* A Python script.


## **Workshop Goals**

By the end, you should understand:

* Why PRs matter and the value of PR templates 
* Why standards exist
* How GitHub helps teams collaborate
* How small habits improve team efficiency


## **Key Message**

> A good repository is not about code, it’s about how a team works together.
